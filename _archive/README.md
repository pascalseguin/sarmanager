# _archive

Legacy files from the Electron + Express + SQLite + Vite era of SAR Manager.

The project pivoted to a pure Next.js web app (no Electron, no Express, no SQLite).
Everything in this folder is kept for reference only — none of it is loaded or used by the current app.

## Contents

| Path | What it was |
|---|---|
| `components/` | React components that were never wired into the current Next.js app |
| `docs/BUGS.md` | Bug tracker for the old Electron app |
| `docs/MIGRATION_PLAN.md` | Plan to migrate from Electron to Next.js (now complete) |
| `docs/blueprint-archived.md` | Spec doc from the Electron era |
| `docs/ARCHIVED.md` | Old feature archive notes |
| `media/` | Screenshots and diagrams of the old Electron app |
| `logs/error` | Browser error log dump (Overpass API 406s) |
| `feedback claude v1.mp4` | Session recording |
