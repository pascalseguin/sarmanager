# SAR Manager

## Description

This program is a SAR Manager application designed to augment and consolidate all the tools that SAR managers are already using in Alberta. Currently, these tools are located in separate places, and this program aims to provide a unified platform where a SAR manager can log in and manage all these tools from one central location without having to switch between them.

## Features

- Centralized login and management interface
- Integration of existing SAR tools
- Streamlined workflow for SAR operations
